# Hello world PWA

## Generate test certificates
```
npm install
./scripts/test-certs-gen-root.sh
./scripts/test-certs-gen.sh
```
## Build and run
```
npm install
npm run build
npm run start
```
## Deploy to GitHub pages
```
npm install
npm run build
npm run deploy
```
Open at http://spamfro.github.io/hello-world-pwa/
